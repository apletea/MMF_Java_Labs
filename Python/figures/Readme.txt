Документация для бедных.
Данный шедевр разработки умеет считать ответы для данных фигур(требуемые данные):
1) Куб (ребро)
2) Шар (радиус)
3) Эллипсоид (три полуоси)
4) Тор (только внешний и внутренний радиусы, радиус окружости не нужен)
5) Прямоугольный параллелепипед (длина, ширина, высота)
6) Цилиндр (радиус основания, высота)
7) Конус (радиус основания, высота)

Допускается ввод и других фигур, в таком случае будет выведено сообщение в output.txt о том, что данная фигура неизвестна.
Также можно вводить некорректные данные для нормальных фигур, соответствующее сообщение тоже появится в output.txt
Ввести некорректные данные очень легко, система чувствительна буквально ко всему, кроме регистра. Образец нормального и ненормального ввода можно глянуть в input.txt
